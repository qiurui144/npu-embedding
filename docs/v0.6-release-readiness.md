## 3. 已知 Known Limitations（v0.6 不修，进 release notes）

- breadcrumb 是 item 顶层路径（item 第一个 chunk 的章节），不是具体命中 chunk 的章节路径 — W5+ 透传精确路径
- chunk_offset_start/end 是 sidecar char 累计，不严格对齐原文 — 前端 Reader 跳转 W3v1 仅顶层导航
- web_search_cache 30 天 TTL 但**无清理 route**（`Store::clear_web_search_cache` fn 已实现未挂 HTTP）— W4 加 `DELETE /api/v1/web_search_cache` + Settings UI clear 按钮
- G2 v1 仅计 high_engagement 不创建占位 item — W5/6 G3 才会真正抓内容
- R18 #2 dist build 在沙盒外手动触发；CI release pipeline 应 codified
